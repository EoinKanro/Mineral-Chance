<h2>Mod Description</h2>
<p style="text-align:center"><a href="https://serilum.com/" rel="nofollow"><img src="https://github.com/Serilum/.cdn/raw/main/description/header/header.png" alt="" width="838" height="400"></a></p>
<p style="text-align:center"><a href="https://curseforge.com/members/serilum/projects" rel="nofollow"><img src="https://raw.githubusercontent.com/Serilum/.data-workflow/main/badges/svg/curseforge.svg" width="200"></a> <a href="https://modrinth.com/user/Serilum" rel="nofollow"><img src="https://raw.githubusercontent.com/Serilum/.data-workflow/main/badges/svg/modrinth.svg" width="200"></a> <a href="https://patreon.com/serilum" rel="nofollow"><img src="https://raw.githubusercontent.com/Serilum/.data-workflow/main/badges/svg/patreon.svg" width="200"></a> <a href="https://youtube.com/@serilum" rel="nofollow"><img src="https://raw.githubusercontent.com/Serilum/.data-workflow/main/badges/svg/youtube.svg" width="200"></a></p>
<p><strong><span style="font-size:24px">Requires the library mod&nbsp;<a style="font-size:24px" href="https://curseforge.com/minecraft/mc-mods/collective" rel="nofollow">Collective</a>.</span></strong><strong>&nbsp;<br><br> &nbsp; &nbsp;This mod is part of <span style="color:#008000"><a style="color:#008000" href="https://curseforge.com/minecraft/modpacks/the-vanilla-experience" rel="nofollow">The Vanilla Experience</a></span>.</strong><br><span style="font-size:18px">Mineral Chance is a mod created to make mining a little bit more exciting. It adds a chance to find a random mineral after mining a stone-type block. In the overworld these stone blocks are <strong><em>stone, andesite, diorite, granite, tuff and deepslate</em></strong>. In the nether it's <em><strong>netherrack, basalt and blackstone</strong></em>.<br><br>By default there is a 1/50 chance to find a mineral. The overworld minerals consist of <em><strong>diamonds, iron nuggets, gold nuggets, lapis lazuli, redstone and emeralds</strong></em>. The nether minerals consist of <em><strong>quartz, golden nuggets and netherite scraps(1.16+)</strong></em>. Mining with Silk Touch will disable mineral drops.<br><br>All minerals have the same chance of dropping. The message when finding a random mineral can be changed and turned off in the config.<br></span><br><br><span style="font-size:24px"><strong><span style="font-size:20px">Configurable:</span> <span style="color:#008000;font-size:14px"><a style="color:#008000" href="https://github.com/Serilum/.information/wiki/how-to-configure-mods" rel="nofollow">(&nbsp;how do I configure?&nbsp;)</a></span></strong></span><strong><br></strong><span style="font-size:12px"><strong>extraMineralChanceOnOverworldStoneBreak</strong>&nbsp;(default = 0.02, min 0, max 1.0): The chance a mineral is dropped when an overworld stone block is broken. By default 1/50.<br><strong>extraMineralChanceOnNetherStoneBreak</strong>&nbsp;(default = 0.01, min 0, max 1.0): The chance a mineral is dropped when a nether stone block is broken. By default 1/100.</span><br><span style="font-size:12px"><strong>enableOverworldMinerals</strong>&nbsp;(default = true): If enabled, mining overworld stone blocks in the overworld has a chance to drop an overworld mineral. These consist of diamonds, gold nuggets, iron nuggets, lapis lazuli, redstone and emeralds.</span><br><span style="font-size:12px"><strong>enableNetherMinerals</strong>&nbsp;(default = true): If enabled, mining nether stone blocks in the nether has a chance to drop a nether mineral. These consist of quartz, gold nuggets and netherite scraps(1.16+).<br><strong>disableMineralDropsWithSilkTouch</strong> (default = true): If enabled, minerals won't drop from stone mined with the silk touch enchantment.</span><br><br><span style="font-size:12px"><strong>sendMessageOnMineralFind</strong>&nbsp;(default = true): If enabled, sends a message when a mineral is found to the player who broke the stone block.</span><br><span style="font-size:12px"><strong>foundMineralMessage</strong>&nbsp;(default = "You've found a mineral hidden in the block!"): The message sent to the player who found a hidden mineral when 'sendMessageOnMineralFind' is enabled.<br><br><strong>ignoreFakePlayers</strong>&nbsp;(default = true): If enabled, minerals won't be dropped if the player is a fake. For example when a mod breaks a block as a simulated player.</span><br><br><br><picture><img src="https://github.com/Serilum/.cdn/raw/main/projects/mineral-chance/a.jpg" width="1142" height="606"></picture><br><br><br><span style="font-size:18px"><strong>The effect in action in the overworld with '<em>extraMineralChanceOnOverworldStoneBreak</em>' set to 0.5:</strong></span></p>
